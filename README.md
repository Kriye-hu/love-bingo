# Project11-love-bingo 💕 恋爱刮刮 Bingo

**一个浪漫的双人互动 Bingo 网页游戏**，5×5 格子包含 24 个心动任务 + 1 个免费（今日限定心动位）。
刮开任意一行/一列/对角线即触发 Bingo 奖励券！

## 功能特性

- ✨ **刮刮乐交互** — 点击格子"刮开"解锁任务
- 💾 **进度持久化** — 使用 localStorage 保存刮开进度，关闭页面后回来继续
- 📤 **分享功能** — 一键分享给TA（支持 Web Share API & 剪贴板）
- 🔄 **重置游戏** — 随时重新开始
- 🎊 **Bingo 彩带庆祝** — 完成连线触发彩带动画 + 随机奖励券
- 📱 **移动端适配** — 触摸友好，响应式设计
